mini tweet bot app features

home page
  • translate: choose between ascii art or L337, to receive a string of your translated text
  • shared twitter account for:
    (1) a club, organization, or company
    (2) photo spot at tourist destination or event
    (3) public access for anyone --and their friends!
features page
  • retweet & follow: (1) bot searches linked account with search terms from user input; (2) attempts a retweet; (3) attempts to follow the user of the retweeted tweet; (4) This process is repeated until a tweet is successfully tweeted, or the list of tweets if fully parsed;
  • follow followers: (1) bot follows all your followers that are not yet being followed;
  • follow X more: (1) bot searches linked account with search terms from user input; (2) attempts to follow X new followers from the users in the search results; (3) if there are no results, or all the users are already being followed, then no user is followed.
  • auto retweet: (1) bot searches linked account with search terms from user input; (2) attempts to retweet a tweet; (3) repeats this process every N seconds; (4) repeats for Y iterations
  • auto tweet file: (1) bot tweets a line form the input text file; (2) automation continues every N seconds until EOF (3) file must have .txt extension (4) bot will only tweet 140 chars from each line.
    (5) click here for example file
  • NOTE: if your tweets are not going through, it may be a result of twitter API restrictions or your query from your search terms may return no results, and thus not automate your request. To read more about twitter API, check out the blog, linked below.
  • mini tweet bot blog: read more details about how mini tweet bot works and troubleshooting issues at the blog.
