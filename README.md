# twitter-bot

* **language:** python 3.6.1
* **libraries:** tweepy, time

## Description

This Repo has my (David John Coleman II) integration of the tweepy library
to auto generate tweets, retweets, and to follow users on a cloudfoundry app
with IBM Bluemix.
